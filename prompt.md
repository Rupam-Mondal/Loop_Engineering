## Prompting Technique

1. Zero shot prompting -- (give everything directly)
   example :- "what is 2 + 2 ?"

2. Few shot prompting -- (direct instruction but with some examples)
    examples : "
                what is 2 + 2 ?
                examples:
                    what is 5 + 4?
                    expected output: 9(nine)

                    what is 5 + 5?
                    expected output: 10(ten)
                "

3. Chain of Tought Prompting -- Instruction hey model have some break down of problem and then solve it
